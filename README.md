# Chrome Dark Blue Theme

A dark blue Chrome theme

## Installation

**Step 1:**
```bash
npm i
```

**Step: 2:**
```bash
node index.js
```

**Step: 2:**
```bash
node index.js
```

**Step: 3:**
* Open Chrome extension manager: [chrome://extensions/](chrome://extensions/)

* Drag & Drop theme.zip into extension page
